# GDMC26-GoldenCrow

## Description

## Build your environment

- Install Minecraft 1.21.11
- Download and install [Fabric](https://fabricmc.net/use/installer/) for Minecraft
- Put the `mods` folder into your Minecraft directory (`./minecraft/mods`)
- you can check the API Docs in [gdmc_http_interface](https://github.com/Niels-NTG/gdmc_http_interface) repo

## Usage

```bash
/buildarea set ~-128 0 ~-128 ~128 255 ~128
```
